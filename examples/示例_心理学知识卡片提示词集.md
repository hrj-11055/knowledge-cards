每条提示词都包含完整的风格描述，但你可以在 ChatGPT 对话开头先发一次「风格锚定」消息，后续生成会更稳定：

```

1. 温暖治愈的插画手绘风（参考小红书插画博主、Behance illustration 风格）
2. 整体配色温暖柔和：米白/奶白底色 + 莫兰迪色系点缀
3. 必须包含与心理学主题相关的人物或场景插画（小人、大脑、抽象意象等）
4. 排版简洁美观，留白舒适，字体清晰
5. 所有文字必须是简体中文，字符准确无误，不出现错别字、拼写错误或乱码
6. 16:9 横版构图，文字与插画各占合理比例（推荐左文右图或上图下文）
7. 风格统一，便于做成系列卡片

每张卡片我会单独发送一条结构化提示词，请严格按照内容输出。明白请回复"明白"。
```

# 第 02 章 · 心理的神经生理机制

## 卡片 2.1 · 神经元结构与功能

```
请生成一张 16:9 横版知识卡片，温暖插画手绘风，左文右图版式。

【主标题】神经元结构与功能
【副标题】Neuron · 神经系统的基本单位

【卡片需呈现的中文文字内容（必须准确清晰）】
· 神经元 = 胞体 + 树突 + 轴突
· 胞体（soma）：代谢中心，含细胞核
· 树突（dendrite）：接收信号，短而多分支
· 轴突（axon）：传导信号，长而少分支，外裹髓鞘
· 突触（synapse）：神经元之间的间隙，通过神经递质传递信号
· 传导方向：树突 → 胞体 → 轴突 → 突触（单向传导）

【视觉元素】
右侧画一个卡通神经元：树突像树枝般展开，胞体是一个圆脸（微笑），轴突像一根长触手伸出去，末端分叉形成突触小体，突触间隙有小圆点代表神经递质。

【配色】米白 #FAF6F0 底，主色暖橙 #E8A87C（树突），辅色雾蓝 #B4C7D9（轴突），文字深灰 #3D3D3D。

要求：16:9 横版，神经元插画要可爱有趣，中文标注准确。
```

## 卡片 2.3 · 神经系统组成

```
请生成一张 16:9 横版知识卡片，插画手绘风，层级/树状结构版式。

【副标题】Nervous System · 中枢 + 外周

【卡片文字内容（中文，准确）】
中枢神经系统（CNS）：
· 脑：大脑、小脑、脑干、间脑
· 脊髓：传导通路、反射中枢

外周神经系统（PNS）：
· 躯体神经系统：控制骨骼肌（随意运动）
· 自主神经系统（ANS）：

  - 副交感神经：放松、恢复、休息和消化

【配色】米白 #FAF6F0 底，主色暖橙 #E8A87C，辅色雾蓝 #B4C7D9，文字深灰 #3D3D3D。

要求：16:9 横版，层级关系清晰，中文准确无误。
```
